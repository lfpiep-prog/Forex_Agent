# How to Get Your Twelve Data API Key

To get free Forex data, we use **Twelve Data**.

1.  **Sign Up**: Go to [Twelve Data](https://twelvedata.com/) and create a free account ("Basic Free").
2.  **Get Key**:
    -   Go to your **Dashboard**.
    -   Copy your **API Key** (a long string of numbers/letters).
3.  **Configure Agent**:
    -   Open `.env`.
    -   Add: `TWELVEDATA_API_KEY=dein_key_hier`.
